# GenLab AI Auto Calling Assistant

A production-ready web application that captures leads via a modern landing page and automatically triggers an AI-powered conversational phone call.

## Features
- **Modern Landing Page**: High-conversion UI built with Vanilla JS/CSS.
- **Instant AI Calling**: Integrated with Twilio Voice API.
- **Grok AI Intelligence**: Personalized sales scripts generated by xAI's Grok.
- **Lead Capture**: Automatic lead storage in SQLite.
- **Direct Orchestration**: Pure Python/FastAPI - no complex automation tools required.

## Project Structure
```text
genlab-ai-calling-assistant/
├── backend/            # FastAPI routes and services
├── frontend/           # Landing page assets
├── database/           # SQLite schema and scripts
├── config/             # Pydantic settings
└── main.py             # Entry point
```

## Setup Instructions

### 1. Prerequisites
- Python 3.10+
- Twilio Account (Account SID, Auth Token, and a verified Twilio Phone Number)
- xAI API Key (Grok)

### 2. Installation
```bash
# Navigate to the project directory
cd genlab-ai-calling-assistant

# Install dependencies
pip install -r requirements.txt
```

### 3. Configuration
Rename `.env.example` to `.env` and fill in your credentials:
```env
TWILIO_ACCOUNT_SID=your_sid
TWILIO_AUTH_TOKEN=your_token
TWILIO_PHONE_NUMBER=your_number
GROK_API_KEY=your_grok_key
BASE_URL=https://your-public-url.ngrok-free.app
```

### 4. Running Locally (with ngrok)
Since Twilio needs a public URL to send the AI script, you must use ngrok:

1. In one terminal, start the app:
   ```bash
   python main.py
   ```
2. In a second terminal, expose it:
   ```bash
   ngrok http 8000
   ```
3. Copy the ngrok URL (e.g., `https://random.ngrok-free.app`) and paste it as `BASE_URL` in your `.env`.

### 5. Deployment
- **Backend/Frontend**: This app is consolidated. You can deploy it as a single Docker container on **Render** or **Railway**.
- **Static Files**: The FastAPI server automatically serves the `frontend/` directory at the root URL.

## Architecture
1. **User** submits name/phone on landing page.
2. **FastAPI** saves the lead to `leads.db`.
3. **Twilio service** triggers an outbound call to the user.
4. When user answers, Twilio requests `/ai-call` from the backend.
5. **Grok AI** generates a personalized intro.
6. **Twilio** speaks the AI's words to the user.
